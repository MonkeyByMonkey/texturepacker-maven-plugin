texturepacker:pack

Full name:

pm.monkey.maven.plugins.texturepacker:texturepacker-maven-plugin:0.1.0-SNAPSHOT:pack

Description:

Pack textures with LibGDX's TexturePacker.

Attributes:

Required Parameters

Name Type Since Description
inputDir File - Input directory.
outputDir File - Output directory.
packFileName String - Pack file name.

Optional Parameters

Name Type Since Description
alias boolean - If true, two images that are pixel for pixel the same will only be packed once.
Default value is: true.
alphaThreshold int - From 0 to 255. Alpha values below this are treated as zero when whitespace is stripped.
Default value is: 0.
bleed boolean - If true, RGB values for transparent pixels are set based on the RGB values of the nearest non-transparent pixels. This prevents filtering artifacts when RGB values are sampled for transparent pixels.
Default value is: true.
combineSubdirectories boolean - If true, the directory containing the settings file and all subdirectories are packed as if they were in the same directory. Any settings files in the subdirectories are ignored.
Default value is: false.
debug boolean - If true, lines are drawn on the output pages to show the packed image bounds.
Default value is: false.
duplicatePadding boolean - (no description)
Default value is: false.
edgePadding boolean - If true, half of the paddingX and paddingY will be used around the edges of the packed texture.
Default value is: true.
fast boolean - If true, the texture packer will not pack as efficiently but will execute much faster.
Default value is: false.
filterMag Texture$TextureFilter - The magnification filter for the texture.
Default value is: Nearest.
filterMin Texture$TextureFilter - The minification filter for the texture.
Default value is: Nearest.
flattenPaths boolean - If true, only image file names are used in the atlas, any subdirectories are not used.
Default value is: false.
forceSquareOutput boolean - Force the output texturepack to be square.
Default value is: false.
format Pixmap$Format - The format the texture will use in-memory.
Default value is: RGBA8888.
ignoreBlankImages boolean - If true, texture packer won't add regions for completely blank images.
Default value is: true.
jpegQuality float - From 0 to 1. The quality setting if outputFormat is "jpg".
Default value is: 0.9.
maxHeight int - The maximum height of output pages. 1024 is safe for all devices. Extremely old devices may have degraded performance over 512.
Default value is: 1024.
maxWidth int - The maximum width of output pages. 1024 is safe for all devices. Extremely old devices may have degraded performance over 512.
Default value is: 1024.
minHeight int - The minimum height of output pages.
Default value is: 16.
minWidth int - The minimum width of output pages.
Default value is: 16.
outputFormat String - The image type for output pages, "png" or "jpg".
Default value is: png.
paddingX int - The number of pixels between packed images on the x-axis.
Default value is: 2.
paddingY int - The number of pixels between packed images on the y-axis.
Default value is: 2.
pot boolean - If true, output pages will have power of two dimensions (required for GL1).
Default value is: true.
premultiplyAlpha boolean - If true, the RGB will be multiplied by the alpha.
Default value is: false.
rotation boolean - If true, TexturePacker will attempt more efficient packing by rotating images 90 degrees. Applications must take special care to draw these regions properly.
Default value is: false.
stripWhitespaceX boolean - If true, blank pixels on the left and right edges of input images will be removed. Applications must take special care to draw these regions properly.
Default value is: false.
stripWhitespaceY boolean - If true, blank pixels on the top and bottom edges of input images will be removed. Applications must take special care to draw these regions properly.
Default value is: false.
useIndexes boolean - If false, image names are used without stripping any image index suffix.
Default value is: true.
wrapX Texture$TextureWrap - The wrap setting in the x direction for the texture.
Default value is: ClampToEdge.
wrapY Texture$TextureWrap - The wrap setting in the y direction for the texture.
Default value is: ClampToEdge.
